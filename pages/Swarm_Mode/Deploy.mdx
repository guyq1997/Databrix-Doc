# Deployment in bwCloud 

Die JD sollte auf eine VM mit docker-compose.yml Datei und andere Konfigurationsdateien eingesetzt werden. Der Einsatz ist ein zweistufiger Prozess:

1.	SSL Zertifikat erstellen
2.	Docker-compose.yml erstellen
3.  Dataien konfigurieren
4.  DHBW IT Helpdesk Ticket erstellen, um Shibboleth SP bei DHBW Shiboleth IdP zu registrieren.
 
**Bemerkung:**
Wie in der Einleitung beschrieben, sollte für eine VM nur eine JD eingesetzt werden. 
Je nach Anzahl der Teilnehmer sollte der Kursanbieter die Anzahl der Gruppen festlegen. Jede Gruppe verwendet dann eine JD (eine VM). 
Z.B. für einen Kurs mit dreißig Nutzern, die in zehn Gruppen aufgeteilt sind, wären beispielsweise zehn VMs erforderlich.
Je nach den für die Aufgabe erforderlichen Rechenressourcen sollte die Größe der VM ausgewählt werden.
