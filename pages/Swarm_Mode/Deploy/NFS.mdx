# Network File System

## Manager Node

``` js copy
sudo apt-get install nfs-kernel-server 
sudo apt-get install nfs-common

nano /etc/exports
/opt/exchange_data ip_worker(rw,no_subtree_check,no_root_squash,sync)

sudo exportfs -ra 
sudo service nfs-kernel-server restart
```

## Worker Node
``` js copy
sudo apt-get install nfs-common
cd /opt
sudo mkdir mount
sudo mount ipadresse_manager:/opt/exchange_data /opt/mount 
```
