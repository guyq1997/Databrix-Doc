# Ihre Programmierumgebung starten und stoppen

Als Administrator des Kurses können Sie Ihre Programmierumgebung jederzeit ein- und ausschalten.
Zuerst müssen Sie SSH zu Ihre VM. Siehe vorherigen Schritt für Details.
Sie müssen dann die folgenden Schritte durchführen.
1.  Öffnen Sie den Ordner, in dem die Programmierumgebung deployed wurde.
    ``` js copy
    cd /opt/jupyter-dhbw
    ```
2.  Für Start führen Sie folgenden Befehl aus.
    ``` js copy
    docker compose up -d
    ```
3.  Für Stop führen Sie folgenden Befehl aus.
    ``` js copy
    docker compose down
    ```
