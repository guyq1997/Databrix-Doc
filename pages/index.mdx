# Data Science Lab 
 
DHBW Stuttgart bietet [Data Science Lernplattform](https://databrix.org) an. 
Praktische Aufgaben und Übungen sind ein wichtiger Bestandteil für Lernprozess. 
Damit die Teilnehmer möglichst frühzeitig praktische Erfahrungen sammeln können, bietet diese Lernplattform ein Data Science Lab.
Es bietet Studierenden eine Cloud-basierte Umgebung für die Datenanalyse.
In diesem Dokument wird vor allem beschrieben, wie Sie Data Science Lab einrichten und auf Ihrem eigenen Server deployen. 

## Distributions

Data Science Lab ist momentan auf Basis bwCloud, da bwCloud derzeit kostenlos ist. Es gibt drei Distributions, Data Science Lab in bwCloud einzusetzen: 
1.	[The Littlest Jupyterhub (TLJH)](/TLJH) ist geeignet, wenn Sie eine kleine Anzahl von Benutzern haben. Änderungen in der Programmierumgebung und Jupyterhub können schnell und einfach realisiert werden.
2.	[Jupyterhub Docker Modus (JD)](/Jupyter_Docker) verpackt Jupyterhub und Single User Server in ein Docker Image. Dadurch kann eine schnelle Bereitstellung auf mehreren Servern erreicht werden, indem ein Docker-Image bereitgestellt wird. Er eignet sich also für kleine und große Zahl von Nutzern.
3.  [Jupyterhub Docker Swarm Modus (zum User Akzptanz Test)](/Swarm_Mode) verteilt Single User Servers in einem Swarm Cluster, wo ein Manager node gesamtes Cluster verwaltet. Sie verfügen über mehrere Worker nodes, um die Rechen Aufgaben auszuführen. Es eignet für große Zahl von Nutzern.

Jeder Einsatz hat seine eigenen Eigenschaften. Daher sollten für das jeweilige Anwendungsszenario unterschiedliche Bereitstellungsmethoden verwendet werden.
