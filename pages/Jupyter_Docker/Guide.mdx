# User Guide für Verwaltung Ihrer Programmierumgebung 

Die Programmierumgebung für Ihren Kurs wird in der bwCloud bereitgestellt. 
Dieser Guide erklärt, wie Sie die Programmierumgebung in bwCloud verwalten können. Sie umfasst Folgendes:

1. Ressourcen verwalten über bwCloud Dashboard
2. Einloggen in VM
3. Programmierumgebung starten/stoppen
4. Dozent und Studente zu ihrer Programmierumgebung hinzufügen
