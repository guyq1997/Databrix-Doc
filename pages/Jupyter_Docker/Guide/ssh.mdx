# Einloggen in Manager Node

Die Programmierumgebung für Ihren Kurs wird in der bwCloud bereitgestellt. 
Genauer gesagt, wird sie in virtuellen Machinen (VM) in bwCloud deployed. 
Sie müssen sich zunächst per Remote-Zugang in eine bestimmte VM einloggen, um Ihre Programmierumgebung zu verwalten. 
Jede Gruppe von Studierende werden eine VM verwenden. Sie müssen sich in entsprechende VM einloggen, um die Programmierumgebung von bestimmter Gruppe zu verwalten.

Führen Sie folgende Schritte aus, um sich in VM einzuloggen:

## Voraussetzung:
1. Sie müssen sich bei VM registriert sein. Bitte yuqiang.gu@dhbw-stuttgart.de kontaktieren.
2. Sie müssen Ihren eigenen SSH Schlüssel generieren. Und ihren Public Schlüssel dem Administrator geben. Für generation SSH Schlüssel sehen Sie die [Anleitung](https://www.bw-cloud.org/en/bwcloud_scope/use) von bwCloud.

## Einloggen:
1. Öffnen Sie einen Terminal.
2. Öffnen Sie den Odner in Terminal, wo ihrer privater SSH Schlüssel gespeichert ist.
3. ``` js copy 
   ssh -i [privater Schlüssel] [Ihrer Username]@93eca700-b764-4621-ab7f-de0792aa31ab.ma.bw-cloud-instance.org
   ```
   Sie müssen [privater Schlüssel] und [Ihrer Username] mit echten Daten ersetzen. 
4. geben Sie Ihren Passwort für den privaten Schlüssel ein.
