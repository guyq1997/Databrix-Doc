# Jupyterhub Docker Modus (JD) 

Jupyterhub Docker Modus bedeutet, dass Jupyterhub und andere benötigte Komponenten alle in Docker verpackt sind.
Anders als bei TLJH sind bei Docker alle Installationen und Konfigurationen bereits vorbereitet.
Dank der Tatsache,dass Docker nicht vom Operationssystem abhängig ist.
Alles, was die Benutzer tun müssen, ist, Docker Container auf ihren Servern einzusetzen, und die Konfigurationsdatei an ihre Nutzungsanforderungen anpassen.
Somit ermöglicht JD einen schnelles Deployment.
